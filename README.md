## Descripción del proyecto

Este proyecto consiste en un sistema básico de gestión de restaurante desarrollado en Python utilizando Programación Orientada a Objetos (POO). El sistema permite representar productos disponibles y clientes registrados mediante clases organizadas en diferentes módulos.

## Estructura del proyecto

* modelos/

  * producto.py: contiene la clase Producto encargada de representar los productos del restaurante.
  * cliente.py: contiene la clase Cliente encargada de representar a los clientes.

* servicios/

  * restaurante.py: contiene la clase Restaurante que administra productos y clientes.

* main.py:

  * Archivo principal donde se crean objetos y se ejecutan las funciones del sistema.

## Importancia de la modularización

La modularización permite organizar mejor el código, separar responsabilidades y facilitar el mantenimiento del software. Cada clase cumple una función específica, logrando un programa más ordenado y fácil de comprender.
